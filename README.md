# Jollibee Clicker Game

## Project Overview

The Jollibee Clicker Game is a fun and interactive web-based game where players click on a Jollibee character to score points within a time limit. The game features a friendly user interface and responsive elements, making it enjoyable for users of all ages. 

## Installation

To get started with the Jollibee Clicker Game, follow these simple steps:

1. **Clone the repository**: 
   ```bash
   git clone https://github.com/yourusername/jollibee-clicker-game.git
   cd jollibee-clicker-game
   ```

2. **Open the HTML file**:
   Simply open `index.html` in your preferred web browser to start playing.

## Usage

- Click on the Jollibee image that appears in random locations within the game area.
- Your score will increase with each successful click.
- The game lasts for 30 seconds; try to score as many points as you can within this timeframe.
- After time runs out, an alert will appear displaying your final score.

## Features

- **Click Mechanic**: Score points by clicking on the Jollibee character.
- **Dynamic Movement**: The Jollibee image moves to a random position every second.
- **Countdown Timer**: Players have 30 seconds to achieve their highest score.
- **Responsive Design**: The game works well across various screen sizes.

## Dependencies

This project utilizes the following libraries:

- **Tailwind CSS**: For styling the game interface.
- **Font Awesome**: For using icons within the project.

No additional package dependencies were found in `package.json` since this project is primarily composed of HTML and JavaScript.
